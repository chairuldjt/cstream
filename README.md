# CStream - Chinese Drama Streaming

A modern, premium streaming website for Chinese dramas built with Next.js 15, Tailwind CSS v4, and API integration.

## Features
- **Trending & Latest**: Real-time drama updates.
- **Indonesian Dubbing**: Dedicated section for dubbed content.
- **Modern UI**: Dark mode, glassmorphism design, and smooth animations.
- **Streaming**: Built-in video player supporting HLS and mp4.
- **Search**: Fast drama search functionality.
- **Auto-play**: Automatic next episode playback.

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure
- `app/`: Next.js App Router pages.
- `components/`: Reusable UI components.
- `lib/api.ts`: API integration logic.
- `types/`: TypeScript definitions.
